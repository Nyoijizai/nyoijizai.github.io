启动当前将要处理的事件

作者: zhangWei

请求URL

/event/info/open POST

请求体 application/json

{
	"currentPage":"int //当前页(分页工具类Pageable第一页下标为0)",
	"pageSize":"int //页大小",
	"pageOrders":[{
		"direction":"string //排序方式：asc | desc",
		"property":"string //排序字段"
	}],
	"pageOrdersJSON":"string //字段排序规则列表，要求JSON字符串格式，适用get请求时的字段排序规则\t   格式:[{'direction':'desc','property':'id'}]   进行URL编码后传到后端",
	"id":"int //主键",
	"eventName":"string //事件名称",
	"eventType":"int //事件类型：1 应急救援， 2 演练， 3 培训， 4 其他",
	"eventSource":"int //事件来源：1 WHO， 2 国家卫健委， 3 省卫健委， 4 其他有关部门",
	"maintenance":"int //维修/保养：1 维修， 2 保养",
	"eventStatus":"int //事件状态：1 进行中， 2 已结束",
	"remark":"string //描述",
	"createMan":"int //创建人(登录用户表ID)",
	"createTime":"date //创建时间"
}

返回结果

{
	"code":"int //返回码",
	"message":"string //返回消息",
	"data":{
		"id":"int //主键",
		"eventName":"string //事件名称",
		"eventType":"int //事件类型：1 应急救援， 2 演练， 3 培训， 4 其他",
		"eventSource":"int //事件来源：1 WHO， 2 国家卫健委， 3 省卫健委， 4 其他有关部门",
		"maintenance":"int //维修/保养：1 维修， 2 保养",
		"eventStatus":"int //事件状态：1 进行中， 2 已结束",
		"remark":"string //描述",

		"createTime":"date //创建时间"
	}
}

作者: zhangWei

请求URL

/event/info/list GET

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
currentPage int 否 当前页(分页工具类Pageable第一页下标为0)
pageSize int 否 页大小
pageOrders.direction string 否 排序方式：asc | desc
pageOrders.property string 否 排序字段
pageOrdersJSON string 否 字段排序规则列表，要求JSON字符串格式，适用get请求时的字段排序规则 格式:[{'direction':'desc','property':'id'}] 进行URL编码后传到后端
id int 否 主键
eventName string 否 事件名称
eventType int 否 事件类型：1 应急救援， 2 演练， 3 培训， 4 其他
eventSource int 否 事件来源：1 WHO， 2 国家卫健委， 3 省卫健委， 4 其他有关部门
maintenance int 否 维修/保养：1 维修， 2 保养
eventStatus int 否 事件状态：1 进行中， 2 已结束
remark string 否 描述
createMan int 否 创建人(登录用户表ID)
createTime date 否 创建时间

返回结果

{
	"code":"int //返回码",
	"message":"string //返回消息",
	"data":{
		"currentPage":"int //当前页(分页工具类Pageable第一页下标为0)",
		"pageSize":"int //页大小",
		"pageTotal":"int //总页数",
		"totalNum":"long //总条数",
		"content":[{
			"id":"int //主键",
			"eventName":"string //事件名称",
			"eventType":"int //事件类型：1 应急救援， 2 演练， 3 培训， 4 其他",
			"eventSource":"int //事件来源：1 WHO， 2 国家卫健委， 3 省卫健委， 4 其他有关部门",
			"maintenance":"int //维修/保养：1 维修， 2 保养",
			"eventStatus":"int //事件状态：1 进行中， 2 已结束",
			"remark":"string //描述",
			"createMan":"int //创建人(登录用户表ID)",
			"createTime":"date //创建时间"
		}]
