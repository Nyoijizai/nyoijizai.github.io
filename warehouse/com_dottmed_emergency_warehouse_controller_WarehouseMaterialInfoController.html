参数名 类型 必须 描述

pageSize int 否 页大小
pageOrders.direction string 否 排序方式：asc | desc
pageOrders.property string 否 排序字段
pageOrdersJSON string 否 字段排序规则列表，要求JSON字符串格式，适用get请求时的字段排序规则 格式:[{'direction':'desc','property':'id'}] 进行URL编码后传到后端
id int 否 主键
materialName string 否 物资名称
materialCode string 否 物资编码/RFID
assetCode string 否 企业资产编码，如：华西资产编码
departmentType int 否 科室类型：1 急救/重症， 2 门诊， 3 后勤， 4 指挥， 5 重症， 6 超声， 7 清创， 8 留观， 9 药房， 10 耗材， 11 手术， 12 防疫/隔离， 13 消毒， 14 住院， 15 检验
brand string 否 品牌
specification string 否 规格型号
unit string 否 单位：台、件、个、袋、包等
status int 否 物资状态：1 在库， 2 已出库， 3 待出库， 4 损耗， 5 报废，6 维修，7 保养
isBox int 否 是否装箱：0 未装箱， 1 已装箱
boxCode string 否 箱子编码/箱子RFID
